# Taller de Programación I

Repositorio de apuntes, actividades y recursos de la cursada **Taller de Programación I**.

La materia introduce los fundamentos de la programación mediante Python y desarrolla progresivamente el trabajo con algoritmos, variables, estructuras de control, funciones, listas, archivos y módulos.

## Sitio de la cursada

La versión web está disponible en:

[https://carlosjesusgarcia.github.io/python_1/](https://carlosjesusgarcia.github.io/python_1/)

El sitio puede consultarse desde computadoras, tabletas y teléfonos. También puede instalarse como aplicación web en Android y iPhone.

## Contenidos del programa

La cursada se organiza en ocho unidades:

1. Conceptos básicos de programas y lenguajes de programación.
2. Tipos de datos, variables, expresiones y asignaciones.
3. Control de flujo: estructuras secuenciales, iterativas y condicionales.
4. Funciones.
5. Estructuras de datos básicas: arreglos y listas.
6. Modularidad y parámetros.
7. Archivos de entrada y salida.
8. Módulos.
